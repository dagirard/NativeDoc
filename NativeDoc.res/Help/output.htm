Output customization


NativeDoc allows you to put some HTML tags inside the documentation block.

The supported HTML tags that can be used are:
- a
- table
- tr
- td
- ul
- li
- b
- u
- span
- div
- pre
- code
Example:
/** \file stack myStack
* \brief My Library stack.
*
* \author MySelf
* \version 1.0
*
* Here is the description in <b>bold</b>, <u>underline</u>,<i>italic</i>.
* <pre>Here is something in Monospaced font</pre>
*
* http://www.google.fr <- The following link will be automatically converted to <a href="http://www.google.fr">http://www.google.fr</a>
* <a href="http://www.google.fr">http://www.google.fr</a> <- works too.
*
*
*
*/
Tip: Links do not need to be surrounded by <a>, if the engine sees anything that begins with "http://" or "ftp://", it will be converted to a link.
