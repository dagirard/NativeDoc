Objects documentation block


Objects can also have documentation: for example, this is useful for libraries.

To tell NativeDoc that the documentation block that you set is for an object, put the tag "\file" or "\object" (instead of \command or \function...) or put the name of the object on the first line.

Example:
/** \object stack myStack
* \brief My Library stack.
*
* \author MySelf
* \version 1.0
*
*/
or

Example:
/** stack myStack
* \brief My Library stack.
*
* \author MySelf
* \version 1.0
*
*/
